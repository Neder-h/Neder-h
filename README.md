## Featured Work

### SalesAI — Meeting Insights Hub

An on-premise conversational intelligence platform built for multilingual Tunisian sales meetings.

SalesAI transforms conversations that switch between Tunisian Derja, French, Arabic, and English into:

- searchable transcripts,
- structured commercial insights,
- objections, risks, and next actions,
- client records,
- versioned follow-up drafts,
- offer emails grounded in imported documents,
- and sales analytics.

**Core stack**

`React` `TypeScript` `Express.js` `MongoDB` `Redis` `BullMQ`  
`FastAPI` `Whisper Large v3` `faster-whisper` `NLLB-200` `Gemini`

I also fine-tuned Whisper Large v3 using approximately 18.2 hours of Tunisian Arabic and code-switched speech, with 5,451 training samples and 309 test samples.

> The production source code is private because the platform was developed for a real client and deployed on-premise.
